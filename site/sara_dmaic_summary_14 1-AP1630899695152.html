V2 saves 6.52 hours per DMAIC (40.6 %) compared to doing it without the agent. The savings come from capabilities shipped since V1:

  • Process map drafting with BPMN export to ARIS — replaces manual mapping in the Define phase.
  • Charts Creator companion agent — recommends chart types, generates specs, renders charts for Measure & Control phases.
  • Historical DMAICs retrieval from BELTIX SharePoint — surfaces prior similar investigations.
  • Statistical analysis in written form — describes distributions, trends, before/after differences in plain language.

  • Web search with source-hierarchy guardrails — surfaces supplier recalls, regulatory changes, external context.
  • Session resume from Dataverse — picks up multi-session DMAICs where the user left off.
  • Studio migration with confirmation gates & quality validation in Analyse / Improve / Control phases.
