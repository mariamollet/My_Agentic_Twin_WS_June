Firstly, I do find the agent quite useful—it's providing helpful prompts and suggestions for next steps, which definitely supports the DMAIC process. That said, I've noticed it's still important for us to pause and reflect at each stage, and occasionally revisit or redo certain steps as we progress. 2

1st Complexity Screening

Tools & Systems How many different apps, data sources, or systems would this agent need to connect to?
Scale: Low: 1–2 (e.g. just SharePoint, or just Outlook)  |  Medium: 3–5  |  High: 6+
Data Types What kind of data would the agent work with?
Scale: Low: Structured (SharePoint lists, Excel tables, forms)  |  Medium: Mixed (some structured, some documents or emails)  |  High: Mostly unstructured (PDFs, free text, emails, Teams chats)
Logic & Reasoning Would the agent follow a clear set of rules, or would it need to interpret context and make decisions?
Scale: Low: Rule-based (if X then Y)  |  Medium: Some interpretation needed  |  High: Requires contextual reasoning across multiple inputs
Human Oversight Would someone always review the agent's output before it's used or sent?
Scale: Low: Always reviewed before action  |  Medium: Reviewed for sensitive tasks, autonomous for routine  |  High: Needs to act independently with minimal supervision
Regulatory & Compliance Does this process sit within a regulated or auditable environment?
Scale: Low: No regulatory constraints  |  Medium: Some governance or approval requirements  |  High: Subject to GxP, audit trails, or regulatory reporting
Output: Complexity classification.
Simple = user can build it themselves in Agent Builder.
Medium = co-build with a site Ambassador using Copilot Studio or Agent Builder.
Complex = requires the AI&Data team (pro-code, multi-agent, or cross-system integration).

Detailed Value Assessment

Process What process or task would this agent improve?
Business Area Which part of the business does this sit in?
Users Who would use it, and roughly how many people?
Frequency How often is this task performed: daily, weekly, monthly?
Current Effort How long does it take per person per run?
Automation Potential How much of this work do you think an agent could handle: most of it, about half, or just a small part?
Value Dimensions Beyond time savings, would this also help with any of these?
✓ Fewer errors or less rework
✓ Easier to scale
✓ Lower throughput time
✓ Better user experience
✓ Better consistency or quality
✓ Enables better decisions from data
✓ Supports compliance or audit requirements
Output: Screening summary shared with the programme team for go/no-go decision.
